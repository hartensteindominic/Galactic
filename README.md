# Alien Frontier Skirmish

A lightweight browser game where you explore a hostile alien sector, collect energy shards, and fight off waves of enemy creatures.

## Run locally

```bash
python3 -m http.server 8000
```

Open <http://localhost:8000> in your browser.

## Controls

- Move: `WASD` or arrow keys
- Fire: `Space`
- Restart after defeat: `R`
